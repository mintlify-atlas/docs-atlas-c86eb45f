<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 text-center">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8 text-center">
    Get up and running with MkDowner in minutes
  </p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the MkDowner repository and navigate to the project directory.
      
      ```bash
      git clone https://github.com/UlisesGarcia20/app-markdown.git
      cd app-markdown
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required npm packages for the React application.
      
      ```bash
      npm install
      ```
    </Step>
    
    <Step title="Configure the backend URL">
      Create a `.env` file and set the backend API endpoint.
      
      ```bash
      VITE_API_BASE_URL=http://localhost:5001
      ```
      
      <Note>
        Make sure your Python backend is running on the configured port before converting files.
      </Note>
    </Step>
    
    <Step title="Start the development server">
      Launch the application and start converting documents.
      
      ```bash
      npm run dev
      ```
      
      The application will be available at `http://localhost:5173`. Drag and drop your files to convert them to Markdown!
    </Step>
  </Steps>
</div>
